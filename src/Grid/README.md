Component Description
---------------------

A Nessie Grid wraps child components allowing content to be aligned in grid patterns.

Example Usage
-------------

    <Grid align  = "left"
		<Column>
		   <Tag> First tag </Tag>
		</Column>
		<Column>
		   <Tag> Second tag </Tag>
		</Column>
		<Column>
		   <Tag> Third tag </Tag>
		</Column>
    </Grid>
